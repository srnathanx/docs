### Screen: Main Dashboard

**Purpose**: The dashboard is your central navigation hub, providing quick access to all Xenora features.

#### Dashboard Layout

The dashboard consists of:

1. **Left Sidebar Navigation**
   - Overview section
   - Apps section (Knowledge Base, Chat, External ChatBot, History, Usage)
   - Settings (bottom)
   - Collapsible for more screen space

2. **Main Content Area**
   - Quick access cards for primary features
   - Recent activity
   - Usage summary

3. **Top Bar**
   - Current page title
   - Search functionality
   - User profile menu
   - Theme toggle (light/dark mode)
___
#### Dashboard Cards

When you log in, you'll see four main cards:

##### 1. Knowledge Base Card
- **Icon**: Document icon
- **Title**: Knowledge Base
- **Description**: Manage and organize your knowledge base
- **Action**: Click to view all knowledge bases

##### 2. Chat Card
- **Icon**: Chat bubble icon
- **Title**: Chat
- **Description**: View and manage chat conversations
- **Action**: Click to start chatting with AI

##### 3. Usage Card
- **Icon**: Bar chart icon
- **Title**: Usage
- **Description**: Monitor system usage and analytics
- **Action**: Click to view usage statistics

##### 4. Settings Card
- **Icon**: Gear icon
- **Title**: Settings
- **Description**: Configure application settings
- **Action**: Click to access settings
___
#### Sidebar Navigation

**Overview Section:**
- Dashboard (home icon) — Returns to main dashboard

**Apps Section:**
- Knowledge Base — Manage knowledge sources
- Chat — AI conversation interface
- External ChatBot — Deploy chatbots to websites
- History — View past conversations
- Usage — Analytics and metrics

**Settings Section:**
- Settings — Organization, LLM, Plans, Team, Agents, MCP

#### User Profile Menu

Click the user avatar (top-right) to access:

- **User Information**: Name and email
- **ADMIN Badge**: If you're an administrator
- **Theme**: Switch between light and dark modes
- **Hide Help**: Toggle help features
- **Log out**: Sign out of the application

#### Theme Toggle

Xenora supports both light and dark modes:

1. Click your profile avatar
2. Select **"Theme"**
3. Choose:
   - **Light Mode**: Traditional light interface
   - **Dark Mode**: Eye-friendly dark interface
   - **System**: Matches your operating system preference
