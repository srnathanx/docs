### Starting a Conversation

### Screen: Chat Page

**Purpose**: Full-screen chat interface for conversations with AI.

#### Accessing the Chat

**Method 1: From Dashboard**
1. Click the **"Chat"** card on the dashboard

**Method 2: From Sidebar**
1. Click **"Chat"** in the left sidebar

**Method 3: From Knowledge Base**
1. View a specific KB
2. Click **"Chat with This KB"**
3. Opens chat pre-configured with that KB

#### Chat Screen Layout

**Left Panel (Optional):**
- Chat history sidebar
- Previous conversations
- Search chats

**Center Panel:**
- Main conversation area
- Message history
- Input field at bottom

**Right Panel (Optional):**
- Knowledge Base selector
- Model selector
- References and sources panel

#### Starting Your First Chat

<Steps>

<Step title="Select a Knowledge Base (Optional)">
- Click the KB dropdown at top
- Choose which KB to use for context
- Or select **"No KB"** for general conversation
</Step>

<Step  title="Choose an AI Model (Optional)">
- Click the model dropdown
- Select from configured models (Claude, GPT, Gemini, etc.)
- Default model is pre-selected
</Step>

<Step title="Type Your Message">
- Click in the input field at the bottom
- Type your question or request
- Press **Enter** to send
- Or press **Shift+Enter** to add a new line
</Step>

<Step title="View Response">
- AI response streams in real-time
- Formatting (markdown, code blocks) renders automatically
- References appear in right panel (if applicable)
</Step>
</Steps>
