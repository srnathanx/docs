#### Accessing Usage & Analytics

- Click **"Usage"** in left sidebar
- Or click Usage card on dashboard

#### Dashboard Overview

The usage dashboard provides insights into:

1. **Message Volume**
2. **Knowledge Base Performance**
3. **Model Usage Distribution**
4. **Response Times**
5. **Cost Tracking**
6. **User Activity**

#### Key Metrics

##### **1. Message Volume**

**Chart Type**: Line or bar graph

**Displays:**
- Messages per day/week/month
- Trend over time
- Comparison to previous period

**Filters:**
- Date range selector
- Group by (day, week, month)

**Insights:**
- Peak usage times
- Growth trends
- Seasonal patterns

##### **2. Knowledge Base Usage**

**Chart Type**: Pie or bar chart

**Displays:**
- Which KBs are used most
- Queries per KB
- Success rate per KB

**Use Cases:**
- Identify most valuable KBs
- Find underutilized content
- Optimize KB strategy

##### **3. Model Distribution**

**Chart Type**: Pie chart

**Displays:**
- Percentage of queries per model
- Claude vs GPT vs Gemini usage
- Cost implications

**Insights:**
- Model preferences
- Cost optimization opportunities
- Performance comparisons

##### **4. Response Times**

**Chart Type**: Line graph

**Displays:**
- Average response time in seconds
- P50, P90, P99 percentiles
- Trend over time

**Performance Indicators:**
- Green: < 2s (excellent)
- Yellow: 2-5s (good)
- Red: > 5s (needs attention)

##### **5. Plan Usage**

**Progress Bars:**

Shows current usage vs. plan limits:

- **Messages**: `850 / 1,000 messages this month` (85%)
- **Storage**: `320 MB / 500 MB used` (64%)
- **Active Bots**: `2 / 3 bots` (66%)
- **Knowledge Bases**: `5 / 10 KBs` (50%)

**Color Coding:**
- Green: < 70% used
- Yellow: 70-90% used
- Red: > 90% used (approaching limit)

**Warnings:**
- Alert when approaching limits (80%)
- Critical alert at 95%
- Notification to upgrade plan
