**Purpose**: Access, search, and manage all past conversations.

#### Accessing Chat History

**Method 1: From Sidebar**
- Click **"History"** in left sidebar

**Method 2: From Chat Interface**
- Click history icon in chat header
- Opens sidebar with past chats

#### Screen Layout

**Left Panel: Conversation List**
- All past conversations
- Sorted by date (newest first)
- Shows preview of first message

**Right Panel: Conversation View**
- Full chat history
- All messages and responses
- Can continue conversation

#### Features

**Search:**
- Search bar at top
- Searches message content
- Real-time filtering

**Filters:**
- Filter by Knowledge Base
- Filter by AI model used
- Filter by date range
- Filter by tags (if enabled)

**Sort:**
- Newest first (default)
- Oldest first
- Most active
- By title (alphabetical)

**Conversation Actions:**

**Resume:**
- Click on a conversation
- Opens in chat interface
- Continue where you left off

**Export:**
- Click export icon
- Choose format (TXT, JSON, PDF, Markdown)
- Downloads to computer

**Delete:**
- Click delete icon
- Confirm deletion
- **Warning**: Cannot be undone

**Share** (if enabled):
- Click share icon
- Generates public or team link
- Set permissions (view-only, can-comment)
- Set expiration

#### Conversation Details

When viewing a conversation, you see:

- **Title**: Auto-generated or custom
- **Created**: Date and time started
- **Last Updated**: Most recent message timestamp
- **Message Count**: Total messages exchanged
- **Model Used**: Which AI model(s) were used
- **Knowledge Base**: Which KB (if any) was used
- **Participants**: Who was in conversation (for team chats)
